# RotOpia - Calculateur de Rotation Opiacée

Calculateur professionnel de rotation et conversion d'opiacés, conçu pour aider à la décision clinique.

## Fonctionnalités
- **Multi-sources** : Cumulez plusieurs molécules sources simultanément.
- **MME Automatique** : Calcul du Total Équivalent Morphine PO.
- **Gestion PRN** : Intégration de la consommation d'entredoses sur 24h.
- **Ratios Sécurisés** : Basés sur les recommandations SFAP/AFSOS avec réduction de sécurité asymétrique.
- **Entre-dose Cible** : Calcul automatique de la dose de secours selon les paliers (SC spécifique).
- **Justification Clinique** : Détail complet du raisonnement mathématique affiché.
- **Mode Impression** : Fiche propre et épurée pour le dossier patient.
- **Responsive** : Optimisé pour smartphones et tablettes.

## Installation
Application web statique (HTML/CSS/JS). Aucun serveur requis.
Ouvrez `index.html` dans n'importe quel navigateur.

## Avertissement
Cet outil est une aide à la décision. La responsabilité clinique incombe au prescripteur. Une titration clinique est toujours nécessaire après une rotation.
